# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025), written in Python.

## Running Solutions

To run the code for day XX:

```bash
python -m day XX [input_file]
```

The `input_file` parameter defaults to `input` if not provided.

### Examples

```bash
python -m day 01
python -m day 01 example
```
